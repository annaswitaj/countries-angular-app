## Installation

```plaintext
git clone https://github.com/annaswitaj/countries-angular-app.git
cd countries-angular-app
npm install -g nx
npm install -g pnpm
pnpm install
pnpm start
```

## Tools used

-   nx
-   pnpm
-   Angular 15
-   angular-material
